Presentation

Dans le cadre de notre session d’été nous avons décider de travailler sur une application Wear OS permettant de récupérer des données de divers capteurs, pour cela nous avons utilisé une montre Google Pixel Watch. Le but de ce site web est d’expliquer les capteurs présents sur la montre Google Pixel Watch et de décrire le fonctionnement de certains capteurs, en particulier le capteur de PPG et le capteur Galvanic.

Afin de réaliser notre projet nous avons dans un premier temps effectué une recherche de l’ensemble des capteurs présents sur la montre Google Pixel Watch. Pour cela nous avons créé et exécuter une petite application Wear OS permettant de lister dans la console la liste des capteurs présent dans l’appareil. Nous pouvons observer ci-dessous la liste des capteurs présent dans la montre que nous avons étudiée :
